GitHub Sync

  1. Create free account at github.com
  2. New repo → name contractor-data → Private → Create
  3. Add file data.json with content {}
  4. Settings → Developer settings → Tokens (classic) → Generate → check repo scope
  5. Paste token + repo name in the top bar → Connect
⚠ Keep your GitHub token private — never share it.
✓ Always click ↑ Save after making changes.
✓ On a new device, open this file → enter token → click ↓ Load.
