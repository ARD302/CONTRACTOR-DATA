GitHub Setup

ONE-TIME

Step-by-step setup

  1. Go to github.com and create a free account
  2. Create a new repository — name it contractor-data — set it to Private
  3. Inside the repo, create a file named data.json — paste {} as content and commit
  4. Go to Settings → Developer settings → Personal access tokens → Tokens (classic)
  5. Generate a new token — check the repo scope — copy it
  6. Paste your token and repo name (yourusername/contractor-data) in the bar at the top
  7. Click Connect, then ↓ Load to pull data, ↑ Save to push changes
⚠ Keep your token private — never share it. It gives write access to your GitHub repo.
✓ Your data file (data.json) on GitHub is your database — it saves everything.
✓ Always click ↑ Save after making changes so they sync to GitHub.
